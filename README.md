# MP4_Metadata_Parser

displays basic metadata for MP4 files

for example:

`Type:         Video`  
`Duration:     5`  
`Video Codec:  h264`  
`Audio Codec:  aac`  
`Width:        560`  
`Height:       320`   

to use it first compile it with
`go build main.go`

then run it with
`.\main.exe <path-to-MP4>`
